jsonクラス生成元ファイル(*.dart) -> jsonクラス生成先ファイル(*.g.dart)

jsonクラスを生成する際には以下のコマンドをターミナルから実行する。
>>dart run build_runner build --delete-conflicting-outputs
※実行ディレクトリはプロジェクトのディレクトリでよい
※生成先ファイルはデフォルトで生成元ファイルと同じディレクトリに生成される

また、以下のコマンドを実行することにより生成元ファイルに合わせて自動で生成が行われる
>>dart run build_runner watch --delete-conflicting-outputs
※実行ディレクトリはプロジェクトのディレクトリでよい
※自動生成はVScodeを閉じるとなくなる
